<h1 align:"center">Tasks & Caixa</h1>

##Esse simples sistema foi feito por mim para minha namorada que tem uma papelaria e precisava organizar as finanças e tarefas de forma centralizadas e personalizada.

O sistema armazena as tarefas e registros de movimentações de caixas no localstorage, permitindo que a pagina seja alocada na memória do dispositivo, sem haver a necessidade de uma conexão com banco de dados.
Os dados se manterão integros desde que o usuário não limpe os cookies e cache do navegador, demais ações como reiniciar o dispositivo, fechar a aba ou recarrega-la, não inputarão na perda dos dados.
As funcionalidades de imprimir relatórios e limpar tabelas é uma preventiva para garantir o bom uso do site e evitar a perca de dados sem congestionar a memória local.
